# mapscad

Mapscad is a browser-based CAD tool for turning real-world map areas into 3D-printable terrain. Pick a region, and it samples the elevation and OpenStreetMap features (streets, tracks, buildings, water) into a solid model you can shape — height exaggeration, a socket, tiling, separate coloured objects — and export as STL or 3MF.

**[Try it live →](https://because-why-not.github.io/mapscad)**
